# PROMPTS COGNITIVO PARA GENERAR CONTENIDOS ACADÉMICO

## PROMPT PARA REDACTAR UN ESQUEMA DEL MARCO TEORICO

```text
Actúa como un asesor metodológico. Elabora un esquema lineal para el marco teórico de una tesis sobre el tema: "[TMEA]".
Organiza la información de lo general a lo particular, incluye antecedentes y asegúrate de que los títulos sigan una 
secuencia lógica para una investigación de nivel universitario.
```

## PROMPT PARA BUSCAR ARTÍCULOS ACTUALES DE ALTO IMPACTO

```text
Actúa como un experto en investigación académica. Necesito que busques artículos de investigación empírica publicados en
revistas científicas indexadas, específicamente de los últimos 5 años. El tema central es: "[TEMA]". Proporciona una lista
de al menos 5 artículos relevantes, incluyendo el enlace al texto completo. 
```

## PROMPT PARA RESUMIR ARCHIVOS PDF

```text
Actúa como experto en síntesis de información. Analiza este [PDF] y genera un resumen ejecutivo de máximo 300 palabras.
Debes extraer los objetivos del documento, las ideas principales, los datos más relevantes si los hay, y las conclusiones.
Presenta los puntos en un formato de viñetas para que sea fácil de leer.
```

## 5 PROMPTS DE IA QUE CONVIERTE ARCHIVOS EN VERDADERAS CLASES

```text
-- Prompt 1: ENSÉÑAME DESDE CERO
Transformar este documento en una clase guiada paso a paso. Explica todo comenzando por los conceptos más básicos y avanza gradualmente
hasta el nivel intermedio y avanzado. Asume que es mi primera vez aprendiendo este tema.

-- Prompt 2: AULAS ESTRUCTURADAS POR CAPÍTULO
Divida este documento en partes y transformar cada [sección o capítulo] en una mini-aula independiente. Explica los puntos
principales de forma clara, añade ejemplos prácticos y termina cada parte con un resumen rápido para reforzar el aprendizaje.

-- Prompt 3: SIMPLIFICAR CONTENIDOS DIFÍCILES
Analiza este documento e identifica las partes más completas o confusas. Reeplícalas usando un lenguaje simple, analogías fáciles de enetender
o ejemplos del mundo real para que todo quede más claro.

-- Prompt 4: CONCEPTOS CLAVE Y DEFINICIONES ESENCIALES
Analiza este documento e identifica todas las ideas, términos y nociones fundamentales. Explica cada una de forma clara y directa,
como si necesitara momorizarla para un examen o prueba importante.

-- Prompt 5: MAPA MENTAL AUTOMÁTICO
Transforma el contenido de este documento en un mapa mental estructurado, conectado ideas principales, conceptos, secundarios
y relacionados importantes para facilitar la memorización.
```
